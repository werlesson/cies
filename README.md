## Deploying Project on Firebase

### Install firebase dependences
```
npm install -g firebase-tools
```
### Log in on Firebase Account 
```
firebase login
```
### Build Project
```
npm run build
```
### Deploy Project
```
firebase deploy - m "Deploying the best feature ever"
```
